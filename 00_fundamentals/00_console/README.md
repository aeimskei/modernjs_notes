# Commands to Log in Console

Recommended browser for development is Google Chrome.

```
console.log('Hello World'); // string
console.log(123); // number
console.log(true); // boolean
var greeting = "Hello World";
console.log(greeting); // variable
console.log([1,2,3,4]); // array
console.log({a:1, b:2}); // object
console.table({a:1, b:2}); // table

console.error('This is some error'); // error
console.clear(); // clears the console
console.warn('This is a warning'); // warning
console.time('Hello'); // timer begins
  console.log('Hello World');
  console.log('Hello World');
  console.log('Hello World');
  console.log('Hello World');
  console.log('Hello World');
  console.log('Hello World');
console.timeEnd('Hello'); // timer ends
```
